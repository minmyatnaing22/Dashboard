# StudyFlow

A productivity dashboard built with HTML, CSS, and JavaScript to help students manage tasks, track goals, stay focused with the Pomodoro technique, and monitor study progress.

## Features

### Task Management

* Add, edit, complete, and delete tasks
* Filter tasks by All, Completed, and Pending
* Task progress tracking
* Local storage persistence

### Goal Tracking

* Add up to 5 yearly goals
* Delete goals
* Automatically saved using local storage

### Pomodoro Timer

* Start, pause, and reset timer
* Pomodoro session tracking
* Focus time tracking

### Analytics Dashboard

* Tasks completed statistics
* Completion rate calculation
* Pomodoro session count
* Focus time tracking
* Weekly activity chart
* Analytics reset functionality

### Additional Features

* Motivational quote generator using API
* Dark / Light mode
* Toast notifications
* Responsive layout
* Sidebar navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Fetch API
